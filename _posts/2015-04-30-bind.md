`bind()` was introduced in 2009â€™s ECMAScript 5 (ES5) specification. It can be used to perform a "partial function application". A new function is created that calls the original, and specifies a number of new arguments.

{% highlight js %}
function foo(something) {
  console.log(this.a, something);
  return this.a + something;
}

var obj = {
  a: 2,
};

var bar = foo.bind(obj);

var b = bar(3); // 2 3
console.log(b); // 5
{% endhighlight %}

In an example similar to that above, `bind()` is used to create a new function that is takes `add()`, but only requires one parameter, as y is always 1:

{% highlight js %}
function add(x, y) {
  return x + y;
}

var plus1 = add.bind(null, 1);
console.log(plus1(5)); // 6
{% endhighlight %}
